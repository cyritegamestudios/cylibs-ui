battle_util.is_valid_target (target_id)
Determines if the given target exists and is not dead.

Parameters:

  • target_id number Mob id

Returns:

    Bool True if the target is valid and false otherwise

Determines if the given target exists and is not dead.

Parameters:

  • target_id number Mob id

Returns:

    Bool True if the target is valid and false otherwise
battle_util.target_mob (target_index, engage)
Switches the player's target and engages the given mob. Note that this function sends an action (0x01A) packet.

Parameters:

  • target_index number Mob index
  • engage Boolean If true, the player will also engage the target
battle_util.is_mob_target (mob_index, target_index)
Returns whether a mob's is targeting a target.

Parameters:

  • mob_index number Mob index
  • target_index number Target index

Returns:

    Bool True if the mob's target matches the given target, false otherwise.
