Module Familiar

Base class for a familiar.

Functions

new (familiar_id, action_queue)
Default initializer for a new Avatar.

Parameters:

  • familiar_id number Mob id of the familiar
  • action_queue ActionQueue Action queue

Returns:

    Familiar A familiar
destroy ()
Stops tracking the player's actions and disposes of all registered event handlers.
monitor ()
Starts monitoring the automaton's actions. Note that it is necessary to call this before events will start being triggered. You should call destroy() to clean up listeners when you are done.
is_idle ()
Returns true if the familiar is currently not engaged.

Returns:

    Boolean True if the pet is idle, and false otherwise.
is_engaged ()
Returns true if the familiar is currently engaged.

Returns:

    Boolean True if the pet is engaged, and false otherwise.
leave ()
Releases the familiar.
