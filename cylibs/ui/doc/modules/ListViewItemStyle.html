new (selectedBackgroundColor, defaultBackgroundColor, fontName, fontSize, fontColor, highlightColor, padding, strokeWidth, strokeAlpha, bold)
Creates a new ListViewItemStyle instance.

Parameters:

  • selectedBackgroundColor table The selected background color as {alpha, red, green, blue}.
  • defaultBackgroundColor table The default background color as {alpha, red, green, blue}.
  • fontName string The font name.
  • fontSize number The font size.
  • fontColor table The font color as {red, green, blue}.
  • highlightColor table The highlighted font color as {red, green, blue}.
  • padding number The padding on each side.
  • strokeWidth number The stroke width.
  • strokeAlpha number The stroke alpha.
  • bold boolean Whether the text should be bolded.

Returns:

    ListViewItemStyle The newly created ListViewItemStyle instance.
